## How to contribute to A-ORM

#### **Did you find a bug?**

* **You can open a GitHub issue.

#### **Did you write a patch that fixes a bug?**

* Open a new GitHub pull request with the patch.

* Ensure the PR description clearly describes the problem and solution. Include the in line comment's.

#### **Did you fix whitespace, format code, or make a purely cosmetic patch?**

Changes that are cosmetic in nature and do not add anything substantial to the stability, functionality, or testability of will generally not be accepted.

#### **Do you intend to add a new feature or change an existing one?**

* You can open a GitHub issue.

#### **Do you have questions about the source code?**

* You can open a GitHub issue or send a mail [info@guneskorkmaz.net](mailto:info@guneskorkmaz.net).

Simply if you have anythink idea or improvment open a issue/pull request or mail me.

Thanks!
